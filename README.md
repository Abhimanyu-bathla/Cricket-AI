# 🏏 Cricket-AI

An AI-powered cricket decision support system that recommends optimal bowling strategies using historical IPL match data. The project leverages machine learning to analyze match situations and provide real-time tactical recommendations through a FastAPI backend and interactive frontend.

---

## 🚀 Features

- Predicts optimal bowling strategies
- Machine Learning-based decision support
- FastAPI REST API
- Interactive web interface
- Player performance analytics
- Match-state analysis
- Historical IPL data processing

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Abhimanyu-bathla/Cricket-AI.git
```

Move into the project

```bash
cd Cricket-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the backend

```bash
uvicorn main:app --reload
```
